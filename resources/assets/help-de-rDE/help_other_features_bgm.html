Hintergrundmusik

Nachdem Sie eine Auduodatei hinzugefügt haben, können Sie diese als Hintergrundmusik des Projekts festlegen, indem Sie den Schalter für die Hintergrundmusik auf an stellen. Wenn Sie eine Audiodatei als Hintergrundmusik eingestellt haben, wird diese während des gesamten Projekts wiedergegeben, auch wenn die Quellvideodatei nicht verschoben wird.
