Eine Videoebene hinzufügen

Tippen Sie auf und dann auf , um den Medienbrowser (Video) zu öffnen; wählen Sie dort das Video, das Sie als Ebene hinzufügen möchten.



Eine Bildebene hinzufügen

Tippen Sie auf und dann auf , um den Medienbrowser (Bild) zu öffnen; wählen Sie dort das Bild, das Sie als Ebene hinzufügen möchten.



Eine Sticker-Ebene hinzufügen

Tippen Sie auf und dann auf , um die gewünschten Sticker auszuwählen.



Eine Textebene hinzufügen

Tippen Sie auf und dann auf , um den gewünschten Text für die Ebene einzugeben.



Eine Handschriftebene hinzufügen

Tippen Sie auf und dann auf , um einen handgeschriebenen Text oder eine Zeichnung hinzuzufügen.



Ebeneneinstellungen anpassen

Nachdem Sie mit eine Ebene erstellt haben, tippen Sie auf die Registerkarte und passen Sie die Optionen wie gewünscht an.
