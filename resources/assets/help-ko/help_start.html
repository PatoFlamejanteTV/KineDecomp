프로젝트 생성부터 공유 하기까지

  1. 프로젝트를 생성하기 위해 을 선택하십시오.
  2. 를 선택하십시오.
  3. 프로젝트 제목을 적어주세요.
  4. 미디어 브라우저에서 원하는 비디오 클립들과 이미지를 선택 한 다음 버튼을 선택 하십시오.  
  5. 원하는 테마를 선택하시고 다음 버튼을 선택하십시오. (선택사항) 
  6. 타이틀을 입력하시고 다음 버튼을 선택하십시오. (선택사항)
  7. 원하는 음악을 선택하시고 다음 버튼을 선택하십시오. (선택사항)
  8. 편집한 영상을 저장하거나 공유하려면, 편집화면에서 버튼을 선택하십시오.
  9. 공유하고자 하는 어플리케이션을 선택 하십시오.
  10. 내보내기와 공유 단계를 마치시면 공유 전 화면으로 이동합니다. (※ 구독상품을 선택하시면 편집한 영상의 워터마크를 지울 수 있습니다.)

(내보내기 한 영상은 단말기의 export 폴더(KineMaster/Export)에서 확인할 수 있습니다.)


기존 사용자 가이드

키네마스터를 실행 시키면 기존 프로젝트들이 아래로 스크롤 할 수 있는 목록으로 표시됩니다.

기본적으로 목록 상단에 가장 최근에 편집 한 프로젝트가 배치됩니다. 프로젝트 이름 순으로 정렬을 하시려면 설정에서 변경 할 수 있습니다.

프로젝트에 대한 기능들은 상세화면에서 할 수 있습니다. 프로젝트 목록에서 프로젝트 선택하면 상세 화면으로 이동 할 수 있습니다. 프로젝트 편집 하기 위해서는 버튼 선택 합니다.

재생하시려면 버튼, 공유하기 위해서는 버튼, 삭제를 하시려면 ￼ 버튼을 선택 합니다.


프로젝트 편집 화면의 미리보기 화면 왼쪽에 위치한 버튼 영역이 추가되었습니다. 되돌리기 하시려면 버튼을 다시 설정 하시려면 버튼을 선택 합니다. 버튼을 통해 테마를 변경 할 수 있습니다. 그리고 프로젝트 편집이 완료되면 편집 화면에서 바로 버튼을 통해 공유 할 수 있고, 버튼을 통해 프로젝트 관련 설정을 변경 할 수 있습니다.￼


미리보기 오른쪽에 위치한 버튼 영역에서는 미디어 및 오디오 를 추가하는 버튼을 그대로 사용할 수 있습니다. 그리고 버튼을 통해 음성 녹음은 화면을 보면서 바로 녹음할 수 있습니다. 버튼을 선택하여 캠코더와 카메라로 새로운 비디오와 사진을 찍을 수 있습니다.

기존에 있던 스티커 와 손글씨 를 레이어 로 추가 할 수 있습니다. 타임라인에서는 버튼을 선택해서 프로젝트 시작시간으로 이동합니다. 버튼을 선택 하여 모든 클립을 볼 수 있도록 타임라인을 확장 할 수 있습니다. 버튼을 통해 프로젝트를 재생 할 수 있습니다.

클립을 선택하면 기존과 동일하게 편집 옵션이 표시됩니다. 다만 위치가 미리보기 화면 오른쪽으로 표시됩니다.


클립을 삭제하기 위해서는 클립 선택 후, 미리보기 화면 왼쪽의 버튼을 선택 합니다. 캡처 메뉴는 미리보기 화면 왼쪽의 버튼을 통해 사용 가능 하지만 정지화면 분할 및 삽입 은 트림 / 분활 메뉴로 이동했습니다.

전환 효과 또한 아이콘을 선택 하면 미리보기 화면 오른쪽 영역에서 표시됩니다.

공유기능은 편집화면, 상세화면에서 버튼을 선택하면 됩니다.

공유 버튼 선택 > 무료 버튼 (혹은 구독 옵션 버튼)을 선택 > 공유 할 어플리케이션을 선택 > 해상도 선택 (단말의 성능에 따라 표출되는) 하여 내보내기 & 공유를 진행 합니다.
