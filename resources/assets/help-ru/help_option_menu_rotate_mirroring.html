Зеркальное отображение

Нажмите, чтобы выбрать изображение или видео и нажмите ,чтобы открыть меню поворота, в котором кнопка используется для зеркального отражения по горизонтали, а – для зеркального отражения клипа по вертикали.


Поворот

Выберите нужное изображение или видео и нажмите , чтобы открыть меню поворота. Кнопка используется для вращения клипа против часовой стрелки на 90 градусов, а – для вращения по часовой стрелке клипа на 90 градусов.
